# Turnbound Forge

A Turnbound build planner with a 7×7 grid and tetromino tiles. Plan your builds, place tiles, and share layouts with others.

## Planned Features

- Grid planner (7×7)
- Tetromino tiles
- Rotation
- Unlock-anywhere
- Trinket slots
- Weapon cap validation
- Shareable URLs

## Tech Stack (planned)

- Next.js
- TypeScript
- Tailwind
- Zustand

## Status

Scaffolding in progress.
